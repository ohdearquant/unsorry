Added deterministic proof import narrowing (ADR-074 / SPEC-074-A): after a proof verifies locally, `swarm/agent.sh` runs `tools.proof.min_imports` to replace the prover's broad `import Mathlib` with the smallest known import set for that proof family (e.g. `Mathlib.Data.ZMod.Basic` + `Mathlib.Tactic` for ZMod divisibility proofs), **re-verifies** it (`lake build --wfail` + axiom audit), and falls back to `import Mathlib` byte-for-byte on any failure — so narrowing can never reject a sound proof. Shrinking the mathlib closure cuts Gate A's critical-path axiom-audit step ~281 s → ~141 s (~2x, measured #2397) for covered proofs, and lowers the build/replay cost too. Gated by `UNSORRY_MIN_IMPORTS` (default on; set `0` to disable). Goal modules (`goals/*.lean`, ADR-018) are never touched.
