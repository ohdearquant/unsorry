Added **`tools/seedkit/`**, a contributor tooling kit that generates, validates, and queues batches of kernel-verified `M ∣ nᵃ − nᵇ` divisibility theorems (a finite `ZMod M` `decide` lifted to `ℤ` through `ZMod.intCast_zmod_eq_zero_iff_dvd`, clean axioms — no `native_decide`). Each goal is proven true before any file is written and is run through triviality, Gate A (`lake build --wfail`) and Gate B before one `queued/prove/*` branch per goal is pushed (push-only; the dispatcher opens and auto-merges). Ships narrow and widened generators, the 5-file artifact writer, and batch/pool drivers with a build timeout; the pipeline is family-agnostic and reusable for other proof families. Docs in `tools/seedkit/README.md`.
