ADR-038/SPEC-038-A: a **shared web-surface design language**, documented once as the canonical visual base for every browser-facing surface (the #270 leaderboard and the #371 proof-graph visualiser) so they read as one product — soft off-white canvas, a centred white `rounded-3xl` card, Inter typography, the brand palette (gold/silver/bronze + blue accent + slate text/muted), a header pattern (light "Unsorry" wordmark + status chip + summary stat chips + cross-link to the sibling surface), and shared badge/state-panel/scrollbar conventions. The tokens are deliberately re-declared per surface rather than shared via a build step, keeping the generated visualiser decoupled from the hand-maintained leaderboard page.
