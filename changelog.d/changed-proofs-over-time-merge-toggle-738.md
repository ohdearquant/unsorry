The leaderboard's **Proofs over time** chart now offers a *solve / merge* basis toggle. The new default — **merge time, bucketed hourly** — plots each proof at the moment its `prove(…)` PR landed on `main`, so a wave of merges clearing an earlier solve backlog reads as recent slope rather than a flat tail (previously the line keyed only on the recorded AISP solve date, which collapses a busy merge day onto the day the proofs were actually solved). The chart gained a true time-proportional x-axis and a faint per-bucket throughput overlay so cadence is explicit and a single spike no longer flattens everything after it; the original daily *solve* view stays one click away.
