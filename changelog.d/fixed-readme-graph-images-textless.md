Fixed the two graph images on the README homepage that rendered without any text. They pointed at the external swarm app's OG endpoints (`swarm.unsorry.agentics.org.nz/api/og/leaderboard` and `/api/og/proofs-over-time`), which emit PNGs whose labels silently drop when the renderer can't load its font — so the chart drew but the title, contributor names, and axis labels were missing. They now use the repository's own generated OG cards (`docs/leaderboard.svg`, `docs/proofs-over-time.svg`, regenerated hourly by the leaderboard workflow), which render their text via an SVG `system-ui` font fallback. The click-through links to the live swarm boards are unchanged.
