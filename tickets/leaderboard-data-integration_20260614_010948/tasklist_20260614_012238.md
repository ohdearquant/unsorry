## Final State

Unsorry has a deterministic contributor leaderboard that is generated from committed proof telemetry and verified proof metadata.

The final user-facing flow:

1. Coordinated proof work writes source telemetry into `library/index/*.aisp` and `proof-runs/*.aisp`.
2. `tools.leaderboard` aggregates `goals/*.aisp`, `library/index/*.aisp`, and `proof-runs/*.aisp`.
3. `python3 -m tools.leaderboard --write .` refreshes every generated leaderboard artifact.
4. `python3 -m tools.leaderboard --check .` detects drift for every generated leaderboard artifact.
5. `docs/metrics/community-stats.json` remains the core stats output.
6. `docs/metrics/leaderboard-ui.json` becomes the stable browser data contract.
7. `docs/leaderboard.html` renders real contributor rows from `leaderboard-ui.json`.
8. README/docs link to the leaderboard, and a generated SVG preview can be embedded.

## Decisions Already Made

- Source of truth is the existing in-repo telemetry and `tools.leaderboard.generate.base_stats(root)`.
- Do not infer historical solvers from git authorship, PR mergers, squash commits, or guesses.
- Do not use leaderboard values in Gate A, Gate B, proof admission, or queue selection.
- Do not call the GitHub API for names or avatars.
- Use deterministic contributor links:
  - profile: `https://github.com/<solver>`
  - avatar: `https://github.com/<solver>.png?size=96`
- Canonical rank follows ADR-023:
  - higher verified proof count;
  - then higher difficulty points;
  - then deterministic tie-breaker.
- Display score is for bar length and visual gamification:
  - `score = difficulty_points * 100 + verified_proofs * 25`
- Failed attempts are visible but not score-positive.
- Local-only smoke runs and infrastructure failures before a real provider attempt do not create leaderboard telemetry.
- README image rendering is included as a pure generated SVG preview task.
- Browser screenshot PNG rendering is out of scope for this ticket unless explicitly re-scoped later; it adds headless-browser dependency and determinism risk.
